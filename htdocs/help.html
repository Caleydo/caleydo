Prerequisites

Caleydo visualizes thousands of data points and therefore is a resource intensive application and requires somewhat recent hardware. The system requirements vary widely with the amount of data you want to load and visualize simultaneously. We recommend the following hardware and software configurations:

  • A graphics card with at least 64 MB RAM is recommended.
  • Up to date graphics card drivers
  • Currently only Sun Java 6 is supported. Download and install the latest Java Runtime Environment. OpenJDK is not supported
  • Supported operating systems: Windows and Linux. Mac Operating Systems are not supported (though it should theoretically run).
  • A mouse with a mouse wheel.
  • For the first start only: a broadband Internet connection.

Installation

Webstart intallation

To run Caleydo as a webstart application, make sure you have Java installed and then simply click the large web start icon on the home or the downloads page.

The application now downloads to your local hard drive and then executes. Note that you do not need to have administrator privileges to do that.

Using web start ensures that you always have the most current version of Caleydo.

Pathway Fetching

When you start Caleydo for the first time you will be asked to fetch the pathways from BioCarta and KEGG. Click Fetch pathways to start the process. Caleydo will then load the pathways from the databases and store them on your local computer.

The download will not surpass 30MB, but will exceed 1000 files. The data is stored in your users home directory.

When the download is complete click Finish. You will need to do this only once, but you can redo it to update the pathway database.

Startup

Upon startup you are presented with choices for different execution modes:

  • Create new project
  • Pathway viewer mode
  • Start with sample gene expression data

Select Create new project if you have your own gene expression data that you want to load. Select the Pathway viewer mode if you don't have gene expression data but are interested in browsing pathways and their interconnections. Select Start with sample gene expression data if you want to try out Caleydo without bothering loading your own data. The data you see in this mode is randomly generated.

Data Loading

To be able to import gene expression data in Caleydo you have to follow these conventions:

  • The first column of the file has to contain a valid RefSeq accession number.
  • Data for one microarray experiment is in one column and all columns are of equal length.
  • If you have not-a-number values, they have to be specified by NaN.
  • Every experiment has a caption
  • The file has to be comma separated text file. The delimiter is arbitrary (can be selected upon import).
  • You may have columns in the file which you do not want to import.
  • Your file may contain a header which is to be ignored.

To select a data file click the Choose data file button. If you have a header in your file, adjust the number of ignored lines by editing the Ignore lines in header text field.

The delimiter of your file can be adjusted by checking the appropriate box for the separators. The data preview will adjust immediately.

If you want your data to be logarithmized for the visualizations check the log 10 box.

The min and max options are disabled by default. If you want to clip your data to a certain range use min and max to do that. Note that you currently have to use both or none for it to work.

To import data you have to select the type for each column. Currently the first column has to be the RefSeq list. Therefore you have to select RefSeq at the first drop down box. The rest should be experiments. If you want to ignore a line simply don't select it.

Uninstallation

When you want to uninstall Caleydo you need to run javaws -viewer in a console, which will bring up the Java web start management console. In the Java Cache Viewer select Caleydo and remove it. Now switch to Resources in the cache viewer and delete all Caleydo resources as well.

Finally, go to your home directory (C:\Documents and Settings\User\ in Windows or /home/user/ in Linux) and delete the .caleydo folder.

Known Issues


Known bugs are listed in our bug tracking system.
Please help us to improve Caleydo by creating a new bug report.

Windows

Many issues on Windows are due to old Java versions or old graphics card drivers. Please update those if possible.
  • Web start windows are blank
    When starting via web start the download window and the security alert window are blank (render the background). This is an issue with graphics card drivers.
    Solution: Install the latest graphics card drivers.
