Projects

The Caleydo Project consists of three main parts: the Framework, Genetical Data Analysis and Clinical Data Analysis

Subprojects

  • Caleydo Visualization Framework
    The framework provides the core functionality for the other projects. It provides data handling, event management and view management. Through its architecture it supports a rapid prototyping approach when creating and integrating new visualizations.

  • Pathway Exploration
    The Pathway Exploration project tries to facilitate the understanding of pathways, which are graphs that model biological processes. These pathways are hand-crafted and therefore have a good layout, but are hard to navigate. With this project we try to overcome this limitation.

  • Gene Expression Analysis
    The Gene Expression Analysis is concerned with the visual analysis of gene expression regulation data, as is retrieved from microarray experiments. We incorporate and enhance modern InfoVis techniques to present the data and to augment the information in pathways.

  • Medical Glyphs
    Medical glyphs visualize different clinical parameters, e.g. age, staging, survival data etc. and their value distributions. Glyphs have symmetry axes (like a crystal) in order to provide an efficient way of spatial arrangement for the interactive definition of hierarchical structures and subgroups. Within a visual data mining application medical glyphs can be arranged according to different spatial grammars and support contextual views on huge data sets.

  • Visual ICD-10 Codes
    Development of medical symbols, which can be used in visual representations of medical records, and to build complex medical glyphs. Each of the basic symbols is available in different abstraction levels.
