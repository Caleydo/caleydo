Caleydo Visualization Framework

Caleydo is a multi-purpose visualization framework which supports state-of-the-art visualization techniques.

Used Technologies

  • Java
    Caleydo is developed using Java in the version 1.5 and 6.

  • JOGL
    The Java OpenGL library is used to render the information in 3D views.

  • RCP
    The Rich Client Platform is a powerful framework developed by the Eclipse Foundation to support the creation of Java applications. RCP based applications provide features as tabbing, view management, preference management, update mechanism and many more..

  • SWT
    Caleydo uses the Standard Widget Toolkit for painting 2D GUI elements. Also RCP bases on the SWT library.

Visualization Techniques

  • Focus + Context
  • Details on demand
  • Linking and Brushing
  • Visual Links

Usage

Caleydo is a research project.
Therefore we are open for external partners who want to use or contribute to the framework. For research institutions it is also possible to get access to the source repositories. Feel free to contact us under caleydo@icg.tugraz.at
