Searching for genes and pathways

The search view can either be opened by clicking on the lens icon in the general tool bar or via Search > Search Gene/Pathway.

In the sample screenshot below the user searches for the string "can". The system provides all entities (pathways and genes) in which the search term is found. The search is non-case sensitive. Below the search field the user can include/exclude data tables in which the search is performed.
The output is a list of genes (upper table) and a list of pathways (lower table). An additional pathway table (bottom right) presents pathways in which the currently selected gene occurs. The gray highlight in the gene table marks names / IDs in which the search term was found.

The context menu provides all actions that can be triggered on the current selection. The following actions can be performed via the gene context menu:

  • Open in browser
  • Load containing pathways in Bucket
  • Show gene in heat map
  • Show gene in parallel coordinates
For pathways these actions are available:
  • Open in browser
  • Load pathway in Bucket
