Uninstallation

Webstart

The webstart version of Caleydo does not change your registry, or writes any start menues. However, if you want to remove all the files associoated with Caleydo, you need to run javaws -viewer in a console, which will bring up the Java web start management console. In the Java Cache Viewer select Caleydo and remove it. Now switch to Resources in the cache viewer and delete all Caleydo resources as well.

Standalone

To remove the standalone versions of Caleydo simply remove the caleydo folder.

Removing the preferences

The go to your home directory (C:\Documents and Settings\User\ in Windows or /home/user/ in Linux) and delete the .caleydo folder.
