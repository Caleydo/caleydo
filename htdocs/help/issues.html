Known Issues

General

Windows

Many issues on Windows are due to old Java versions or old graphics card drivers. Please update those if possible.
  • Web start windows are blank
    When starting via web start the download window and the security alert window are blank (render the background). This is an issue with graphics card drivers.
    Solution: Install the latest graphics card drivers.
