Version 1.1 Beta

New Features

  • New Hierarchical Heat Map that allows to browse all genes
  • New Tabular Data View that shows you your data in a spreadsheet - which is synchronized with the other views
  • Many newly designed elements
  • New export functionality that enables you to save and re-read results of your analysis
  • New method for browsing a set of selected genes by propagating them and viewing them in a "list heat map" next to the bucket and the parallel coordinates
  • New way of applying random sampling: only the displayed elements are sampled, all operations are executed on the whole dataset. Also the sample automatically adapts to the number of visible element resulting in all elements being visible once a selection was narrowed down.
  • New user interface that improves clarity
  • New navigation features in the Bucket including drag and drop and removing of views.
  • Re-ordering, duplicating and removing of experiments is now synchronized across all views.
  • Drag and drop for re-ordering axes in the parallel coordinates
  • New global brush in the parallel coordinates that allows you to filter out all genes that never leave a threshold
  • New way to handle NaN values in the parallel coordinates
  • Experiment selection now possible in heat map

Version 1.0.2

New Features

  • Views have a new "reset view" function that clears all changes
  • Genes in pool layer are highlighted as well
  • Added log base 2 support
  • Added an XP classic style mode. (Enable via Preferences -> General)
  • Added support for different filter levels (all genes - genes with david mapping - genes that occur in pathways). All genes mode is slow, needs improvement.
  • Data loader is now a view, not only a dialog. This allows runtime changes to data.

Fixed Bugs

  • Color mapping now uses correct values on reset default
  • Default color for NaN set to gray
  • Improved mouse events under windows, needs further testing
  • Min and max works now separately
  • Pathways can be loaded more than once now
  • Selection consistency when moving views improved
  • Data RefSeq mapping loader ignores version of RefSeqs (NM_*.* -> NM_*).
  • Multiple RefSeq numbers per micro-array spot in data files are ignored because they all point to the same DAVID gene ID.
  • All experiments in data loader are selected by default.
  • Corrected mapping of genes to actual values
  • Browser now does not complain all the time when it is offline
  • All delimiters for data import work now (not only Tab)

Version 1.0.1

  • Added proxy support for pathway download
  • Added NAN (not a number) support
  • Improved connection lines by using curved edge bundling
  • Improved look and feel

  • Initial release version
