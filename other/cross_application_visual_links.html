Visual Linking ist eine Methode der InfoVis, bei der Relationen zwischen Entitaeten in verschiedenen Views mittels gezeichneter Linien verdeutlicht werden. Traditionell wurden Relationen zwischen Multiple Linked Views durch Highlighting der verschiedenen Elemente erzielt. Visual Links erweitern diesen Ansatz indem sie direkte Verbindungen zwischen den Elementen zeichnen, und somit den "Cognitive Load", also den Assoziationsaufwand, fuer den Benutzer reduzieren.

Visual Links sind im Caleydo Framework bereits realisiert. Allerdings sind diese auf ein Fenster beschraenkt. Ziel der vorgeschlagenen Arbeit ist es, Visual Links ueber Applikationsgrenzen hinweg zu zeichnen. Das Ergebniss soll sich am Mock-Up das im Bild zu sehen ist orientieren.

Die technische Umsetzung erfolgt mittels eines Compositing Window Managers fuer Linux (Compiz Fusion). Applikationen die das Visual Linking unterstuetzen sollen, koennen angepasst werden. Je nach Umfang der Projektes (Magisterpraktikum oder Diplomarbeit) kann auch ein nicht in die Applikation eingreifendes, auf Bilderkennung basierendes Interface erstellt werden.

Die Arbeit erfolgt in Koopoeration mit den Caleydo und Deskotheque Projektteams.

Der Schwerpunkt der Arbeit liegt darin eine Prototypen, fuer Cross-Application Visual Links zwischen dem Caleydo Framework und einer anderen Applikation, wie einem PDF Reader oder einem Web-Browser zu implementieren
