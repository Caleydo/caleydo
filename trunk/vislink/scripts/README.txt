Usage: 

Set environment variables for Deskotheque and Caleydo root folders: 
$ export DKTROOT=/home/you/where_deskotheque_is_located/src
$ export CALEYDOROOT=/home/you/where_caleydo_is_located

Run the script dktJava: 
$ ./sliceVisRenderer.sh

There should now be a folder
$CALEYDOROOT/vislink/generated_src/VIS
with the Java interfaces for the VisRenderer.
